
# Human Resource Analytics Dashboard

An interactive Human Resource Analytics Dashboard built using Microsoft Excel to analyze employee data, attrition trends, salary distribution, employee satisfaction, and workforce performance.

---

## 📌 Project Overview

This project provides HR insights through interactive charts, KPI cards, and filters.  
The dashboard helps organizations monitor employee performance, attrition, salary patterns, and department-wise workforce analysis.

---

## 📊 Dashboard Highlights

### KPI Metrics
- Total Employees: 1200
- Total Attrition %: 48%
- Average Monthly Salary: 68,559
- Average Age: 41
- Average Job Satisfaction: 3.0
- Overtime Employees: 53%

---

## 📈 Visualizations Included

### 1. Employee by Department
Displays total employees across:
- Sales
- Operations
- Marketing
- HR
- IT
- Finance

### 2. Attrition by Department
Shows employee attrition count department-wise.

### 3. Performance Rating
Performance distribution categorized as:
- Excellent
- Good
- Average
- Low

### 4. Gender Distribution
Visual representation of male and female employee ratio.

### 5. Average Salary by Job Role
Compares salary distribution among departments.

### 6. Attrition vs Job Satisfaction
Analyzes relationship between employee attrition and satisfaction score.
